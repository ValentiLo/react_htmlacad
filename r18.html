Пример 1

Третий параметр метода React.createElement принимает содержимое элемента. Мы уже передавали в свойство строки, но это могут быть и дочерние элементы.

Добавим в заголовок элемент span с классом text-green, который изменяет цвет текста.Класс надо создать в стилях.

Например, так 

          <style >
        .text-green {
            color: green;
            font-weight: bold;
            background: yellow; /* Добавим фон для наглядности */
            padding: 5px;
        }
        
        .text-underline {
            text-decoration: underline;
            color: blue;
            border: 2px solid red; /* Граница для видимости */
            padding: 10px;
        }
          </style>

React предоставляет нам единообразный механизм для создания разных элементов и управления их свойствами и содержимым.
