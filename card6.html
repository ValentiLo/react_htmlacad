Чтобы карточка могла отрисовать разные слова в разметке, добавим для компонента параметр text. Описание функции изменится на CardWord({text})

Компонент получит от родителя нужный текст, который нам нужно поместить в разметку. Для этого снова воспользуемся синтаксисом для выражений в JSX: <span>{text}</span>. В карточке с картинкой мы использовали фигурные скобки для передачи значения переменной в атрибут элемента, а сейчас — для установки содержимого элемента из переменной.

Теперь можно передать из игрового поля разные слова в карточку, например, так: <CardWord text="Kannettava"/>.
