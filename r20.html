React. Основы синтаксиса

До сих пор мы создавали элементы с единственным элементом-потомком. Но что, если мы захотим создать блок-контейнер и положить в него заголовок и абзац?

Чтобы создать несколько элементов в содержимом, мы можем передать их следующими параметрами: четвёртым, пятым и так далее.

    <script>
      const container = document.getElementById('root');
      const root = ReactDOM.createRoot(container);
      function MyComponent () {
        const title = React.createElement('h1', {className: 'title'}, 'Hello React!');
        const text = React.createElement('p', null, 'Можно передать несколько потомков списком.');
        const container = React.createElement(
          'div', {className: 'container'}, title, text
        );
        return container;
      }
      const element = React.createElement(MyComponent);
      root.render(element);
  </script>

Корневой элемент приложения. Всем, что внутри него, будет управлять библиотека React.
