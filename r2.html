Знакомство с React начнём с основ — с подключения библиотеки. Подключить React к проекту можно несколькими способами.

Самый привычный — установить из npm.

Другой вариант — создать каркас приложения с помощью инструмента Create React App.

В демонстрации мы воспользуемся третьим способом — подключим React в виде обычного JS-модуля. Прямо на страницу. Этот способ хорошо подходит для изучения возможностей библиотеки, но для продуктовой разработки следует выбрать один из предложенных выше вариантов.

Для работы нам потребуются библиотеки React, React DOM и Babel.
