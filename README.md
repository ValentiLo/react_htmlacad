# 1.Разработка компонентов
## Работа над заданием

Загрузите архив с проектом в формате zip и отправьте его на проверку. Максимальный размер архива — 20 МБ.


## Задача текущего этапа — создать первые компоненты и передать им данные.

*Материалы*

**Шаблон проекта, который включает:**

-главный файл проекта index.html;

-готовую вёрстку игрового поля template.html;

-файл с данными для отрисовки карточек data.js;

-библиотеки React.js, ReactDOM и Babel.js (папка vendor).

# Что нужно сделать

- Подключить библиотеки.

- Отрисовать поле с карточками по набору данных.
  
## Порядок действий

- Подключите к странице библиотеки React, ReactDOM и Babel.

- Подключите к странице файл данных.

- Создайте корневой компонент приложения и перенесите в него разметку из шаблона. Разметку оформите по правилам синтаксиса JSX.

- Выделите карточку в компонент, передайте ему данные из приложения.

- Определите из вёрстки шаблона, какие классы отвечают за отображение состояния карточки.

- Добавьте в компонент карточки условие для добавления классов.

- Добавьте параметр key для элементов списка.

- Используйте готовые списки выделенных и отгаданных карточек для отладки компонентов.

# 2.Разработка интерактивных компонентов


## Задача текущего этапа — организовать поток данных от пользователя.

## Материалы

**Обновлённый шаблон проекта*, который включает:**

- новые стили для счётчика шагов (css/style.css);
  
- обновлённый файл с данными data.js (мы убрали из данных наборы выделенных карточек).
  
## Что нужно сделать

- Открывать карточки по клику пользователя.

- Посчитать ходы.
  
# Порядок действий

- Доработайте компонент игрового поля (сетки карточек), определите список выделенных карточек в константе.
  
- Добавьте состояние компоненту сетки и обработчик события onClick элементу карточки.

- Запишите идентификатор карточки в список открытых при клике на ней. Список хранится в состоянии компонента.

- Добавьте проверку в обработчик события, чтобы не выбирать повторно уже выбранные и отгаданные карточки.

- Доработайте компонент игрового экрана и объявите в его состоянии счётчик шагов. Отобразите значение счётчика в компоненте.

- Увеличьте счётчик шагов на 1 при каждом открытии карточки.
