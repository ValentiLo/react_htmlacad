React. Основы синтаксиса

В синтаксисе JSX имя тега преобразуется в первый параметр метода React.createElement. Пользовательские компоненты обязательно должны начинаться с заглавной буквы.

Значения атрибутов после имени тега будут собраны в объект и переданы вторым параметром в React.createElement. Например, <h1 className="title"> после преобразования даст React.createElement('h1', {className: 'title'}).

В третий параметр React.createElement попадает содержимое тега между угловыми скобками. Например, <p>Содержимое тега</p> будет преобразовано в React.createElement('p', null, 'Содержимое тега').

        <script type="text/babel">
            const container = document.getElementById('root');
            const root = ReactDOM.createRoot(container);
            function MyComponent () {
              const title = <h1 className="title">JSX</h1>;
              const text = <p>Упрощает написание кода.</p>;
              return React.createElement(
                'div', {className: 'container'}, title, text
              );
            }
            root.render(<MyComponent/>);
        </script>
