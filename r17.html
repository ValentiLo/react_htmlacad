React. Основы синтаксиса

Пожалуй, самое используемое свойство элемента — это набор его классов. Вы уже умеете работать со списком классов методами classList.add и classList.remove. Для работы с полным значением атрибута class у элемента есть свойство className. Это именно свойство, а не метод. Чтобы установить значение some-css-class атрибуту class, мы должны записать это значение в свойство className. Например, так: element.className = "some-css-class".

Метод React.createElement предоставляет нам удобный механизм для декларативного указания атрибутов элемента. Передадим при вызове вторым параметром объект, и React сам установит нужные значения свойств.
