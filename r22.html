React. Основы синтаксиса

Мы уже увидели, что React упрощает и унифицирует создание элементов. Но даже с несколькими элементами видно, что вызовы React.createElement занимают больше места в коде, чем сами параметры элементов.

Чтобы упростить код, мы воспользуемся синтаксисом JSX. Браузер работает только со стандартным синтаксисом, поэтому нам нужно подключить библиотеку Babel.js и сказать браузеру, что содержимое тега <script> будет преобразовано.

Библиотека Babel найдёт на странице «свой» скрипт с type="text/babel" и заменит в нём все конструкции вида <MyComponent/> на React.createElement(MyComponent).

         <script type="text/babel" >
            const container = document.getElementById('root');
            const root = ReactDOM.createRoot(container);
            function MyComponent () {
              const title = React.createElement('h1', {className: 'title'}, 'Hello React!');
              const text = React.createElement('p', null, 'Используем синтаксис JSX.');
              const container = React.createElement(
                'div', {className: 'container'}, [title, text]
              );
              return container;
            }

            // Это равносильно строкам ниже:
            // const element = React.createElement(MyComponent);
            root.render(element);
          </script>
