Чтобы научить карточку отображать картинку с нужным адресом, добавим параметр для компонента. Для этого в функции компонента объявим параметр url, и функция будет выглядеть так: CardImage({url}).

Компонент карточки должен установить значение параметра для атрибута src картинки. В JSX есть специальный синтаксис для выражений — фигурные скобки. Запись src={url} означает, что нужно прочитать значение переменной url и записать его в атрибут src.

Теперь мы можем передать адрес картинки в карточку из списка, например, так: <CardImage url="img/photo-card-1.jpg"/>.
