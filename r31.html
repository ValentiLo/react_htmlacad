React. Основы синтаксиса

Пользовательские компоненты часто требуют дочерних элементов для отрисовки в собственной разметке. Для этого в синтаксисе JSX предусмотрена более удобная запись. Всё, что будет указано между открывающим и закрывающим тегами пользовательского элемента, автоматически попадает в параметр children.

Чтобы получить содержимое для пользовательского компонента, достаточно объявить параметр функции: function MyComponent ({children}).

Так мы можем описать структуру вложенности с пользовательскими компонентами как для обычных HTML-элементов.
