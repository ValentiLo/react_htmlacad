React. Основы синтаксиса

Вам уже знакомы методы document.createElement и append (или appendChild), которые используются для создания и размещения элементов на странице.

React предоставляет свой механизм для создания элементов — метод React.createElement. При вызове первым параметром мы передаём тип элемента, вторым — его свойства, а затем содержимое элемента.

Создадим заголовок и для отрисовки на странице передадим его методу root.render.
