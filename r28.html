React. Основы синтаксиса

Напомним, атрибуты JSX-элемента будут переданы вторым параметром в метод React.createElement и записаны в свойства созданного DOM-элемента.

При создании пользовательского компонента значения атрибутов будут переданы в параметры функции-декларации компонента. Задача компонента — указать значения параметров вложенным элементам. Без явного указания на корневом элементе className={className} название класса для <MyComponent className="container"/> «потеряется», это только параметр функции. Мы снова использовали фигурные скобки, чтобы получить значение параметра, а не строку с текстом.

Корневой элемент приложения. Всем, что внутри него, будет управлять библиотека React.
